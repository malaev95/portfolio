Kanban

Dit is een project die ik en mijn medestudent hebben waargemaakt als resultaat van een grondig onderzoek. Het doel was om het Kanban-systeem uit te leggen tijdens een presentatie en verbanden te leggen tussen Kanban en Scrum methode. Dit project heeft ons veel tijd gekost. Omdat het een nieuw onderwerp was moesten we veel opzoekwerk doen. Na het verzamelen van voldoende informatie zijn we van start gegaan met het verdelen van deze informatie in kleine hoofdstukken. Als laatste stap hebben we deze hoofdstukken in 1 of 2 dia's geplaatst. Als besluit kunnen we meedelen dat we veel hebben bijgeleerd van dit project.

Extreme Programming

Een project waarbij men geavanceerde programmeertechnieken moet toepassen om een simpele probleem op te lossen in de Java-taal. Deze technieken konden we zelf kiezen. Ik vond een heleboel op stackoverflow. Deze heb ik dan ook toegepast. Dit project is momenteel volop aan de gang dus kan ik het resultaat niet meteen laten zien.

Sportsmania

Sporters op deze niveau geven niet snel op waardoor het voor de scheidsrechter heel moeiljk wordt om een beslissing te nemen.
